# Screenshots

This folder contains recruiter-facing screenshots of Cash Compass for the GitHub README and project documentation.

## Current Files

- `landing-page.png` - public fintech landing page with the product hero, dashboard preview, demo CTA, and privacy-focused messaging.
- `demo-login.png` - seeded demo login flow for quick recruiter review.
- `dashboard.png` - main dashboard with balance, income, expenses, savings, charts, quick actions, bills, and goals.
- `transactions-ledger.png` - unified ledger with income, expenses, bills, recurring rows, search, filtering, and management controls.
- `csv-upload.png` - manual statement upload page with privacy copy, upload preview area, and validation-focused workflow.
- `analytics-or-assistant.png` - analytics view with chart-based finance review and spending breakdowns.

## When To Update

Update these screenshots when a visible page layout changes, branding changes, major workflows are added, or the README no longer matches the deployed app.

Recommended source: capture the deployed app at `https://cash-compass-finance.vercel.app` using the seeded demo workspace. Keep images clear, current, and reasonably compressed without making text blurry.
